> Binary utilities

## Install

With [npm](https://npmjs.org/) installed, run

```
$ npm install -g @ninestuff/bin
```

## Usage

After installed it globally, just run `nine`.

## Usage without installing

Just run

```
$ npx @ninestuff/bin
```
